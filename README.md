# 🌦️ Weather Dashboard App

A responsive and modern Weather Dashboard web application that provides real-time weather updates using OpenWeather API.

🔗 **Live Demo:**
https://weather-ritishraj.netlify.app/

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💨 Wind speed and humidity details
* ☁️ Dynamic weather icons
* 📱 Responsive design for mobile and desktop
* ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* OpenWeather API
* Netlify

# 🍴 Fork and Clone Repository

## Step 1: Fork the Repository

Click the **Fork** button at the top-right corner of this repository to create your own copy.

---

## Step 2: Clone the Repository

Run the following command in your terminal:

```bash id="e4g8pz"
git clone https://github.com/ritishraj/Weather-Dashboard-App.git
```

---

## Step 3: Navigate to Project Folder

```bash id="k7upxw"
cd Weather-Dashboard-App
```

---

## Step 4: Open Project

You can directly open:

```bash id="l2ryjv"
index.html
```

OR use VS Code Live Server extension.

---

# ⚙️ Installation

No additional dependencies are required.

Simply open the project in your browser.

---

# ▶️ Run Locally

If using VS Code:

1. Install Live Server Extension
2. Right click on `index.html`
3. Click **Open with Live Server**
