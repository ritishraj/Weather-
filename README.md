# 🌦️ Weather Dashboard App

A responsive and modern Weather Dashboard web application that provides real-time weather updates using OpenWeather API.

🔗 **Live Demo:**
https://weather-ritishraj.netlify.app/

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💨 Wind speed and humidity details
* ☁️ Dynamic weather icons
* 📱 Responsive design for mobile and desktop
* ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* OpenWeather API
* Netlify

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ritishraj/Weather-Dashboard-App.git
```

Move to project folder:

```bash
cd Weather-Dashboard-App
```

Run the project:

```bash
Open index.html in browser
```
