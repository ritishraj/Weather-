# 🌦️ Weather Dashboard App

A responsive Weather Dashboard web application that provides real-time weather information using OpenWeather API.

## 🚀 Features

* Search weather by city
* Real-time temperature updates
* Humidity and wind speed
* Responsive UI
* Dynamic weather icons

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeather API
